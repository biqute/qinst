.. _installation:

Installation
============

Qinst can be installed from PyPI

.. code-block:: bash

   pip install qinst

To install Qinst with all the optional packages (experiments, analysis, docs) run

.. code-block:: bash

   pip install qinst[all]

The optional packages are

- ``experiments`` which provides scripts for standard laboratory routines.
- ``analysis`` for code testing.
- ``docs`` for the documentation source and compilation.
